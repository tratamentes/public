Image Shortcode

Demonstration of the Paige image shortcode.

figures images paige shortcodes

4 de janeiro de 2023

3 minutos

Paige provides a paige/image shortcode for displaying an image.

Code:

{{< paige/image src="cover-image-1.webp" >}}

Result:

Code:

{{< paige/image src="cover-image-1.webp" caption="Landscape" width="50%" link="https://github.com/willfaught/paige" >}}

Result:
